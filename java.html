1. Encapsulation

Bundling data and methods that operate on that data within a single unit:

public class Student {
    private String name;    // private field

    // Public getter
    public String getName() {
        return name;
    }
    
    // Public setter
    public void setName(String name) {
        this.name = name;
    }
}

2. Inheritance

Creating new classes that are built upon existing classes:

// Parent class
public class Animal {
    protected String name;
    
    public void eat() {
        System.out.println("Animal is eating");
    }
}

// Child class
public class Dog extends Animal {
    public void bark() {
        System.out.println("Dog is barking");
    }
    
    @Override
    public void eat() {
        System.out.println("Dog is eating");
    }
}

3. Polymorphism

Ability of objects to take on multiple forms:

// Method overloading
public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
    
    public double add(double a, double b) {
        return a + b;
    }
}

// Method overriding (seen in inheritance example)
Animal myDog = new Dog();  // Polymorphic reference
myDog.eat();  // Calls Dog's eat method

4. Abstraction

Hiding complex implementation details and showing only functionality:

// Abstract class
abstract class Shape {
    abstract double getArea();
    abstract double getPerimeter();
}

// Concrete class
class Rectangle extends Shape {
    private double width;
    private double height;
    
    public Rectangle(double width, double height) {
        this.width = width;
        this.height = height;
    }
    
    @Override
    double getArea() {
        return width * height;
    }
    
    @Override
    double getPerimeter() {
        return 2 * (width + height);
    }
}

5. Interfaces

Defining a contract for class behavior:

interface Movable {
    void moveUp();
    void moveDown();
    void moveLeft();
    void moveRight();
}

class Player implements Movable {
    private int x, y;
    
    @Override
    public void moveUp() { y--; }
    
    @Override
    public void moveDown() { y++; }
    
    @Override
    public void moveLeft() { x--; }
    
    @Override
    public void moveRight() { x++; }
}

Using Scanner

What is Scanner?

Scanner is a class in java.util package used to get user input from various sources (keyboard, files, etc).

Basic Usage

import java.util.Scanner;  // Import at top of file

public class InputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        // Reading different types of input
        System.out.print("Enter your name: ");
        String name = scanner.nextLine();     // Reads whole line
        
        System.out.print("Enter your age: ");
        int age = scanner.nextInt();          // Reads integer
        
        System.out.print("Enter your height: ");
        double height = scanner.nextDouble();  // Reads decimal
        
        scanner.close();  // Always close scanner when done
    }
}

Common Pitfalls

  • nextInt() doesn't consume the newline character
  • Always use scanner.nextLine() after scanner.nextInt() to consume leftover newline
  • Handle InputMismatchException for invalid inputs

Best Practice Example

try {
    Scanner scanner = new Scanner(System.in);
    
    System.out.print("Enter a number: ");
    int num = scanner.nextInt();
    scanner.nextLine();  // Consume newline left-over
    
    System.out.print("Enter text: ");
    String text = scanner.nextLine();
    
    scanner.close();
} catch (Exception e) {
    System.out.println("Invalid input!");
}

Key Methods

  • nextLine() - Reads a line of text
  • next() - Reads a single word
  • nextInt() - Reads an integer
  • nextDouble() - Reads a decimal number
  • hasNext() - Checks if more input exists

Scanner with 2D Arrays

Creating Dynamic Tables

Common exam question: Create a table with user-specified dimensions and values.

import java.util.Scanner;

public class TableCreator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        try {
            // Get table dimensions
            System.out.print("Enter number of rows: ");
            int rows = scanner.nextInt();
            
            System.out.print("Enter number of columns: ");
            int cols = scanner.nextInt();
            
            // Create the 2D array
            int[][] table = new int[rows][cols];
            
            // Fill the table
            System.out.println("Enter numbers for the table:");
            for(int i = 0; i < rows; i++) {
                for(int j = 0; j < cols; j++) {
                    System.out.printf("Enter value for position [%d][%d]: ", i, j);
                    table[i][j] = scanner.nextInt();
                }
            }
            
            // Display the table
            System.out.println("\nYour table:");
            for(int i = 0; i < rows; i++) {
                for(int j = 0; j < cols; j++) {
                    System.out.print(table[i][j] + "\t");
                }
                System.out.println();
            }
            
        } catch (Exception e) {
            System.out.println("Invalid input! Please enter numbers only.");
        } finally {
            scanner.close();
        }
    }
}

Example Output:

Enter number of rows: 2
Enter number of columns: 3
Enter numbers for the table:
Enter value for position [0][0]: 1
Enter value for position [0][1]: 2
Enter value for position [0][2]: 3
Enter value for position [1][0]: 4
Enter value for position [1][1]: 5
Enter value for position [1][2]: 6

Your table:
1   2   3
4   5   6

Key Points:

  • Always validate user input
  • Use try-catch for error handling
  • Consider using printf for formatted output
  • Remember to close the Scanner
